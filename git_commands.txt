git remote add <name(origin например)> <url> -> добавление удаленного сервера

git push -u origin <branch> -> связь локальной ветки репо с удалённой веткой(github)

git remote -v -> ссылка на удаленный сервер
git fetch -> скачивает изменения с удаленного сервера, но не применяет локально
git branch -vv -> показывает свзяь локальной и удалённой ветки
